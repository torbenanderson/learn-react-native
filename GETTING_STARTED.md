**Time needed**: 1-2 hours

### Running Your App

```bash
# Install dependencies (first time only)
npm install

# Start the development server
npm start

# Then press:
# 'w' for web browser
# 'i' for iOS simulator (Mac only)
# 'a' for Android emulator
# Or scan QR code with Expo Go app on your phone
```
